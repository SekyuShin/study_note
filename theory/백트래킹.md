# Backtracking
- 해를 구하는 도중 해가 아니여서, 이전으로(Back) 돌아가 해를 찾는 기법
- DFS와 유사하나 다른점은 가지치기를 한다.
	- 가지치기 : 중간에 나와서 모든 경우 탐색 X
- ![[Pasted image 20251101231848.png]]

### 사용법
임의의 집합에서 주어진 기준 대로 원소의 순서를 선택하는 문제를 푸는 데 적합
```python
def 재귀함수(x):
	if 정답 :
		# 정답 출력 or 저장
	else :
		for 받을 수 있는 모든 자식 노드:
			if 유망:
				# 자식노드로 이동
				# 재귀함수 (x+1)
				# 부모노드로 이동
```
