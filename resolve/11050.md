### 예제

```

```

```
10
``` 

### 풀이
- My solution
```python
import sys
N, K = map(int,sys.stdin.readline().strip().split())
num = 1
for i in range(N-K+1,N+1):
    num *= i
deno = 1
for i in range(1,K+1):
    deno *= i

print(int(num/deno))
```
