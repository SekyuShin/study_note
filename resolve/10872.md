### 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.
### 입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.
### 출력
첫째 줄에 N!을 출력한다.
### 제한
- 
### 예제
- 입력
```
10
```
- 출력
```
3628800
``` 
- 입력
```
0
```
- 출력
```
1
``` 
### 풀이
- My solution
```python
N = int(input())
ret = 1
for i in range(1,N+1):
    ret*=i
print(ret)
```
