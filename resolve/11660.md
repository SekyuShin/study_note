### 문제
N×N개의 수가 N×N 크기의 표에 채워져 있다. (x1, y1)부터 (x2, y2)까지 합을 구하는 프로그램을 작성하시오. (x, y)는 x행 y열을 의미한다.

예를 들어, N = 4이고, 표가 아래와 같이 채워져 있는 경우를 살펴보자.

|   |   |   |   |
|---|---|---|---|
|1|2|3|4|
|2|3|4|5|
|3|4|5|6|
|4|5|6|7|

여기서 (2, 2)부터 (3, 4)까지 합을 구하면 3+4+5+4+5+6 = 27이고, (4, 4)부터 (4, 4)까지 합을 구하면 7이다.

표에 채워져 있는 수와 합을 구하는 연산이 주어졌을 때, 이를 처리하는 프로그램을 작성하시오.
### 입력
첫째 줄에 표의 크기 N과 합을 구해야 하는 횟수 M이 주어진다. (1 ≤ N ≤ 1024, 1 ≤ M ≤ 100,000) 둘째 줄부터 N개의 줄에는 표에 채워져 있는 수가 1행부터 차례대로 주어진다. 다음 M개의 줄에는 네 개의 정수 x1, y1, x2, y2 가 주어지며, (x1, y1)부터 (x2, y2)의 합을 구해 출력해야 한다. 표에 채워져 있는 수는 1,000보다 작거나 같은 자연수이다. (x1 ≤ x2, y1 ≤ y2)
### 출력
총 M줄에 걸쳐 (x1, y1)부터 (x2, y2)까지 합을 구해 출력한다.

### 풀이
- My solution
```python
import sys

N,M = map(int,sys.stdin.readline().strip().split())

arr = []

for _ in range(N):
    arr.append(list(map(int,sys.stdin.readline().strip().split())))

ret = [[0]*N for _ in range(N)]
for i in range(N):
    for j in range(N):
        if i == 0 and j == 0:
            ret[i][j] = arr[0][0]
        elif i == 0:
            ret[i][j] = ret[i][j-1]+arr[i][j]
        elif j == 0:
            ret[i][j] = ret[i-1][j]+arr[i][j]
        else:
            ret[i][j] = ret[i][j-1]+ret[i-1][j]+arr[i][j]-ret[i-1][j-1]

for _ in range(M):
    x1,y1,x2,y2 = map(int,sys.stdin.readline().strip().split())
    x1,y1,x2,y2 = x1-1,y1-1,x2-1,y2-1
    result = 0
    if x1 == 0 and y1 == 0:
        result = ret[x2][y2]
    elif x1 == 0:
        result = ret[x2][y2] - ret[x2][y1-1]
    elif y1 == 0:
        result = ret[x2][y2] - ret[x1-1][y2]
    else:
        result = ret[x2][y2]-ret[x1-1][y2]-ret[x2][y1-1]+ret[x1-1][y1-1]
    print(result)
```
