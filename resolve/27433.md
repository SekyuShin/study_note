### 문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.
### 입력
첫째 줄에 정수 N(0 ≤ N ≤ 20)이 주어진다.
### 출력
첫째 줄에 N!을 출력한다.
### 제한
- 
### 예제
- 입력
```
10
```
- 출력
```
3628800
``` 
- 입력
```
0
```
- 출력
```
1
``` 

### 풀이
- My solution
```python
import sys

def factorial(n):
    if n in [0,1]:
        return 1
    else :
        return n*factorial(n-1)
    
N = int(sys.stdin.readline().strip())
print(factorial(N))
```
