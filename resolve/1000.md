[baekjoon_1000](https://www.acmicpc.net/problem/1000)
### 조건
- 시간 : 2초
- 메모리 : 128Mb
### 문제
- 두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.
### 입력
- 첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)
### 출력
- 첫째 줄에 A+B를 출력한다.

### 예제
- 입력
```
1 2
```
- 출력
```
3
``` 

### 풀이
- My solution
```python
AB = input().split(' ')
print(int(AB[0])+int(AB[1]))
```

- Answer
```python
a, b = map(int, input().split())
print(a+b)
```
