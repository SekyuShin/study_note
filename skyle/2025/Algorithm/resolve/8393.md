### 문제
- n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.
### 입력
- 첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.
### 출력
- 1부터 n까지 합을 출력한다.
### 제한
- 
### 예제
- 입력
```
3
```
- 출력
```
6
``` 

### 풀이
- My solution
```python
n = int(input())
sum = 0
for i in range(1,n+1):
    sum+=i
print(sum)
```

```python
n = int(input())
sum = int(n*(n+1)/2)

print(sum)
```
